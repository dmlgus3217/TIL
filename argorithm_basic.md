## 코딩을 결정하는 기본 알고리즘

#### 스텍자료

- 선입후출 후입선출
- 삽입과 삭제
- stack= [] 리스트

#### 큐 (queue) 자료구조

- 먼저 들어온 데이터가 먼저 나가는 형식 (선입후출)

#### 힙Heap의 특징

- 힙은 완전 이지 트리 자료구조
- 힙에서 항상 루트노드를 제거

- 최소 힙(min heap)

  - 루트 노드가 가장 작은 값을 가진다.
  - 따라서 값이 작은 데이터가 우선적으로 제거됨

- 최대 힙 (max heap)
  - 루트노드가 가장 큰 값을 가진다.
  - 따라서 값이 가장 큰 데이터가 우선적으로 제거됨
